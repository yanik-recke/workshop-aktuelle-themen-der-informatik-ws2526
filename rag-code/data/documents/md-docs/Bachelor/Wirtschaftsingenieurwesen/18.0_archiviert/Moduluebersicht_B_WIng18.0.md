## **Bachelor Wirtschaftsingenieurwesen**

#### Anmerkungen

![](_page_4_Picture_2.jpeg)

- 1) Das Auslandssemester ist in diesem Studiengang obligatorisch. Dual Studierende verbringen das entsprechende Semester stattdessen als Praxissemester in ihrem Betrieb.
- 2) Für Studierende, welche ihr Studium zum Sommersemester begonnen haben und die Vertiefungsrichtung Informationsmanagement wählen, wird das Vertiefungsmodul Einführung in die Programmierung wie im Plan abgebildet in das 2. Semester vorgezogen. Dafür findet das Modul Volkswirtschaftslehre anders als abgebildet im 4. statt im 2. Semester statt.
- 3) Die folgenden Leistungen müssen bis zum Ende des 5. Studiensemesters erbracht werden und sind ebenfalls Voraussetzung für die Zulassung zum Auslandssemester. Ohne erfolgreiche Übergangsprüfung erfolgt die Exmatrikulation.

Analysis Übung Analysis Rechnungswesen 1 Chemie, Chemietechnik Praktikum Chemie Technisches Grundpraktikum Einführung in die Betriebswirtschaft

4) Weitere Informationen zu Prüfungstypen und Vorbedingungen zu Prüfungen finden sich im Studienverlaufsplan. Die Inhalte der einzelnen Lehrveranstaltungen sind im Modulhandbuch beschrieben.
